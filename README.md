# wikiquote_fortune

[![Build Status](https://travis-ci.org/Yailm/wikiquote_fortune.svg?branch=master)](https://travis-ci.org/Yailm/wikiquote_fortune)

inspired by the `python-wikiquote` and `wikiquote_fortune`
